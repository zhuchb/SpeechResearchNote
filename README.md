# My speech research note
本人是语音方向在读PhD，记录一些平常做实验踩到的坑，是给我自己做的笔记，也希望帮到大家。不定时记录，踩到坑了就写一点。

I’m currently a PhD student working in speech-related research. This blog is mainly a place for me to record the issues and pitfalls I run into during experiments, a personal notebook of lessons learned along the way. Hopefully, some of these notes will also be helpful to others. I’ll update it from time to time.
